# カップ数の計算
[ここ](https://www.wacoal.jp/advice/contents/post-15.html)の資料を参考に基準を作成。

Iカップ以降は2.5cmごとに基準を作成。Kカップまで定義

Jカップ: 32.5cm内外

-> 31.5 ~ 33.5

Kカップ: 35cm内外
-> 34.0 ~ 36.0

**カップ数計算表**
上記の基準をそのまま使うと各カップ間で連続値にならないため、下記のように計算表を作成。

下限値を含み、上限値は含まない。

AAA: ~ 6.5

AA: 6.5 ~ 9.0 

A: 9.0 ~ 11.5

B: 11.5 ~ 14.0

C: 14.0 ~ 16.5

D: 16.5 ~ 19.0

E: 19.0 ~ 21.5

F: 21.5 ~ 24.0

G: 24.0 ~ 26.5

H: 26.5 ~ 29.0

I: 29.0 ~ 31.5

J: 31.5 ~ 34.0
